Simple Printing API WebUSB

WARNING

THIS SYSTEM DOES NOT REQUIRE SIDE SERVER API INTEGRATION FOR PRINTING, THE SYSTEM ONLY WORKS ON API CLIENT.

A USB API for printing without the need for a third party, users simply communicate between the computer or mobile device and the printer device.
